# zhihu-api
仿知乎 RESTful API

## Tech Stack

- Node.js
- Koa2
- MongoDB
- Postman
- JWT
- 阿里云

## Build And Setup

```bash
# Install dependencies
npm install

# Start the App in production environment
npm run start

# Start the App in development environment
npm run dev
```
